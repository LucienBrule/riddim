== Quick run
. Start the API in dev mode:
[source,bash]
----
./gradlew :modules:app-api:quarkusDev
----
. Send a sound event:
[source,bash]
----
curl -X POST http://localhost:8080/sound/play \
  -H "Content-Type: application/json" \
  -d '{"type": "kick", "parameters": {"gain": 1.0}}'
----
