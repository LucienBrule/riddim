Assets and Artifacts

Build outputs

  • Library jars: modules/<module>/build/libs/<module>-0.1.0-SNAPSHOT.jar

  • Quarkus runner: modules/app-api/build/quarkus-app/quarkus-run.jar

Tracking files

  • ASSETS.md documents verified build and run outputs.

  • ASSETS_DASHBOARD.md is generated by ./scripts/generate_dashboard.sh.
