= Build and Run

[NOTE]
====
AUDIT_ME: Commands here mirror the current Makefile and Gradle setup. Validate on your machine.
====

== Build and test
[source,bash]
----
./gradlew build
./gradlew test
./gradlew verifyAll # Run all build/test/web/docs/docker checks
./gradlew ci        # Alias for verifyAll
----

== Web & Docs
[source,bash]
----
./gradlew webBuild
./gradlew docsBuild
----

== Dev server
[source,bash]
----
./gradlew :modules:app-api:quarkusDev
----

== Docker
[source,bash]
----
./gradlew dockerBuild
----

=== Docker Compose (Runtime)
[source,bash]
----
make docker-up
make docker-ps
make docker-down
----

== Artifact dashboard
[source,bash]
----
./gradlew assetsDashboard
----
